# Attendance System using Mantra MGS 100 Fingerprint Sensor

## Setup Instructions
1. Install Node.js if not installed already.
2. Open terminal/command prompt in the `attendance-system` folder.
3. Run: 
   npm install
4. Start the server:
   npm start
5. Open browser and navigate to:
   - http://localhost:3000/ (for student attendance)
   - http://localhost:3000/login.html (for admin login)
